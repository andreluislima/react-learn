## RENDERIZAÇÃO CONDICIONAL
A renderização condicional de componentes no ReactJs é uma técnica usada para renderizar diferentes elementos ou componentes com base em uma condição.

## O QUE SÃO EVENTOS?
Eventos em React são AÇÕES que são acionadas pelo usuário (Quando o usuario interage com o sistema) ou pelo sistema e que podem ser usadas para manipular o estado do aplicativo ou executar alguma lógica especifica.
Os eventos em React são semelhantes aos eventos em HTML e JS, mas a diferença é que eles são tratados pelo React de uma maneira mais consistente e eficiente.

>> Evemtos são coisas que mudam de valor.

## O QUE SÃO ESTADOS EM REACT?
Os estados em React são DADOS que podem ser armazenados e gerenciados por um componente, controlando seu comportamento e a exibição de dados na interface do usuário.
Eles são armazenados em objetos Js e podem ser atualizados pelo próprio componente ou por outros componentes que têm acesso a ele.
Quando um estado é atualizado, o React re-rendereiza o componente e atualiza a interface do usuário para refletir as alterações.

## HOOK USESTATE
O useState é um hook que permite que você adicione um estado a um componente funcional do React.
Ele é usado para armazenar e atualizar valores que podem ser usados na renderização do componente.
O useState retorna um array com dois elementos:
* 1. O valor atual do estado.
* 2. Uma função para atualizar o estado.
